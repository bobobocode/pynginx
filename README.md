# PyNginx

PyNginx is an Nginx of Python.

More than that, it is also an [Engine](https://github.com/bobobocode/engine.git) of python version.

## What is same with Nginx

* Usage, especially conf file
* Key concepts of source code
```
a. Global Data Cycle
b. Worker Process
c. Event Mechanism
d. HTTP Framework
```

## Note some differences

* Not all the features, not now at least
