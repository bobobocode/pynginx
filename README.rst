PyNginx
=======

PyNginx is nginx of python version, 
which is written from learning 
nginx `<https://github.com/nginx/nginx.git>`_ 
and tornado `<https://github.com/tornadoweb/tornado.git>`_ code.

Usage
-----

You can use it as same as nginx.
But now only the features which are showing in the conf directory 
is supported.

Especially it also supports used as `engine
<https://github.com/bobobocode/engine.git>`_.
